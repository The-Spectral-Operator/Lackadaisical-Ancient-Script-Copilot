## Core Position

UDM20-CM targets **Cypro-Minoan** (c. 1550–1050 BCE), the undeciphered Bronze Age script of Cyprus. It holds a unique position: sandwiched between the partially understood Linear A (its ancestor) and the fully deciphered Cypriot Syllabary (its descendant), making bidirectional comparative analysis exceptionally promising.

---

## Why a Cypro-Minoan-Specific UDM Is Needed

- Unique genealogical position: Linear A → Cypro-Minoan → Cypriot Syllabary
- The Cypriot Syllabary IS deciphered (reads Arcado-Cypriot Greek), providing descendant phonetic values
- Multiple sub-scripts identified (CM1, CM2, CM3) — possibly different languages or registers
- Corpus is small but growing (~250 inscriptions)
- Bronze Age Cyprus was multilingual (Eteocypriot, Greek, possibly Hurrian elements)

## Phase Adaptations

### PHASE 1: Sign Inventory (Adapted)
- Use Ferrara (2012, 2022) updated sign list
- ~96 signs identified (range: 85-110 depending on allograph decisions)
- Separate CM1, CM2, CM3 sub-corpora strictly
- Map signs to Linear A ancestors AND Cypriot Syllabary descendants
- Output: Three-way correspondence table (LA → CM → CS)

### PHASE 2: MEGA-Correlation (Adapted)
- Tier 1 (35% weight): Cypriot Syllabary (known phonetic values, descendant script)
- Tier 2 (30% weight): Linear A (ancestor script, undeciphered but structurally mapped)
- Tier 3 (20% weight): Ugaritic (contemporary, nearby, different script family but same region)
- Tier 4 (15% weight): Hittite/Luwian cuneiform (Anatolian contacts)
- **Key advantage**: Descendant script values provide testable phonetic hypotheses

### PHASE 3: Semantic Clustering (Adapted)
- CM1 (most texts): Administrative? Literary?
- CM2 (Enkomi only): Possibly different language (Hurrian?)
- CM3 (Ugarit finds): Likely same system in international context
- Inscription contexts: clay balls, tablets, bronze objects, pottery marks
- The Enkomi tablets as primary analytical corpus
- Output: Register/language separation before phonetic analysis

### PHASE 4: Phonetic Reconstruction (Adapted)
- Start from Cypriot Syllabary values projected BACKWARD through sign evolution
- Test each CS-derived value against Linear A structural predictions
- Expect: CV (consonant-vowel) syllabary structure (like both ancestor and descendant)
- Identify: signs that changed phonetic value during CM → CS transition
- Output: Hypothetical phonetic grid with confidence levels per sign

### PHASE 5-8: Cultural Integration
- Phase 5: Late Bronze Age Cyprus economy (copper trade, Alashiya identity)
- Phase 6: Multi-ethnic population models (local, Aegean, Levantine, Anatolian)
- Phase 7: Enkomi as administrative center vs. ritual center
- Phase 8: Collapse context (1200-1050 BCE) and script survival

### PHASE 9-12: Language Identification
- Is CM1 Eteocypriot? (Proposed but unconfirmed)
- Is CM2 Hurrian? (Enkomi as cosmopolitan port)
- Does the script encode one language or multiple?
- Test against known Eteocypriot inscriptions (later, alphabetic)
- Substrate words in Cypriot Greek as potential cognates

### PHASE 13-16: Validation
- Every proposed phonetic value must be consistent across CM1, CM2, CM3 (or explained if not)
- Descendant script values provide hard constraints: CS values that don't work must be flagged
- Archaeological context must be compatible with proposed readings
- New finds from Hala Sultan Tekke and other sites as blind tests

### PHASE 17-20: Breakthrough Tracking
- Currently at: ~10-15% decipherment (sign identifications via CS projection)
- Major opportunity: The Cypriot Syllabary bridge is the strongest in undeciphered script studies
- Barrier: Small corpus + possible multilingual complexity
- Watch for: New tablets from ongoing Cypriot excavations (Hala Sultan Tekke)

## Critical Warnings

1. **Sub-Script Conflation**: CM1/CM2/CM3 may encode DIFFERENT LANGUAGES — do not mix analyses
2. **Descendant ≠ Identical**: Cypriot Syllabary values evolved; they're hypotheses, not certainties for CM
3. **Small Corpus**: 250 inscriptions is borderline for statistical methods
4. **Multilingual Trap**: Don't assume one language — Bronze Age Cyprus was cosmopolitan
5. **Publication Gap**: Many inscriptions still unpublished or in fragmentary preliminary reports
