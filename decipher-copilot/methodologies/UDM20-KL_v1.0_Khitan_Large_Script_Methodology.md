# UDM20-KL v1.0
## Universal Decipherment Methodology v20 — Khitan Large Script Adaptation
### By: Lackadaisical Security - Linguistics Division

---

## Core Position

UDM20-KL targets the **Khitan Large Script** (c. 920–1125 CE), created for the Khitan language of the Liao dynasty in northern China/Mongolia. While the related Khitan Small Script has seen more decipherment progress, the Large Script (logographic, ~3,000+ characters) remains mostly undeciphered. The Khitan language itself is partially reconstructed (Mongolic family, possibly para-Mongolic).

---

## Phase Adaptations

### PHASE 1: Sign Inventory
- ~3,000+ characters (logographic, modeled on Chinese)
- Many characters borrowed directly from Chinese with modified readings
- Epitaph inscriptions provide the primary corpus (~40 known)
- Some characters identifiable via Chinese character components
- Output: Character catalog with Chinese-origin vs. Khitan-innovation classification

### PHASE 2: MEGA-Correlation Analysis (85+ Scripts)
- Tier 1 (35% weight): Regional/contemporary — Chinese characters (the model script), Khitan Small Script (same language, partially deciphered), Jurchen script
- Tier 2 (25% weight): Cultural contact scripts — Uyghur script, Tibetan, Korean (contemporary regional scripts)
- Tier 3 (20% weight): Operator scripts (if available; redistribute if not)
- Tier 4 (20% weight): Proven methodology scripts — approaches used for Jurchen/Tangut decipherment (similar sinographic adaptations)
- **Key**: Chinese original characters + Khitan Small Script readings provide dual anchoring

### PHASE 3: Linguistic Framework
- Language: Khitan (para-Mongolic or early Mongolic)
- Middle Mongolian provides closest comparative data
- Dagur language may preserve Khitan-like features
- Expected: SOV word order, agglutinative morphology, vowel harmony
- Chinese loanwords in Khitan provide confirmed vocabulary points
- Output: Linguistic model from para-Mongolic reconstruction

### PHASE 4-8: Dual-Script Strategy
- The Khitan Small Script (alphabetic/syllabic blocks) partially reads the SAME language
- Use Small Script decipherment progress to inform Large Script readings
- Bilingual Chinese-Khitan epitaphs exist (MAJOR anchor)
- Royal genealogies in both scripts provide name verification
- The Liao dynasty Chinese historical records provide contextual framework

### PHASE 9-12: Character Analysis
- Characters borrowed from Chinese: test if meaning or sound was borrowed
- Characters modified from Chinese: trace the modification logic
- Novel Khitan characters: use positional/contextual analysis
- Compound character formation rules (like Chinese radicals)
- Name transcriptions as phonetic evidence

### PHASE 13-20: Status
- Currently at: ~15-25% (names, titles, some vocabulary via Small Script bridge)
- Best path: Small Script → Large Script transfer + Chinese bilingual analysis
- Barrier: Complex logographic system, limited corpus
- Watch for: New epitaph discoveries in Inner Mongolia/Northeast China
