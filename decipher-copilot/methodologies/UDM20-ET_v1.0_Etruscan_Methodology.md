## Core Position

UDM20-ET targets the remaining undeciphered portions of **Etruscan** (c. 700–100 BCE). The script IS fully read (adapted Greek alphabet), and basic grammar/vocabulary (~250 words) are understood, but the vast majority of vocabulary and complex syntax remain unknown. Like Meroitic, this is a LANGUAGE problem, not a script problem.

---

## Why an Etruscan-Specific UDM Is Needed

- Script fully readable (Greek-derived alphabet)
- Language is an ISOLATE — no confirmed relatives (Rhaetic and Lemnian are close but also poorly attested)
- ~13,000 inscriptions exist but most are very short (funerary)
- Only a few longer texts exist (Liber Linteus, Tabula Capuana, Pyrgi Tablets)
- The Pyrgi Tablets provide a BILINGUAL (Etruscan/Phoenician) — major anchor
- ~250-300 words definitively understood; thousands remain unknown
- Roman-era glosses provide scattered vocabulary confirmations

## Phase Adaptations

### PHASE 1: Corpus Assembly (Script = SOLVED)
- Alphabet known: 26-letter adapted Greek alphabet, right-to-left
- All inscriptions can be phonetically read
- Corpus: TLE (Thesaurus Linguae Etruscae) as primary reference
- Separate by text type: funerary, votive, legal, literary, boundary stones
- The Liber Linteus (Zagreb mummy wrapping) = longest text (~1,200 words)
- Output: Complete digitized corpus with text-type classification

### PHASE 2: MEGA-Correlation Analysis (85+ Scripts)
- Tier 1 (35% weight): Regional/contemporary scripts — Latin, Oscan, Umbrian (Italic neighbors with borrowed Etruscan vocabulary)
- Tier 2 (25% weight): Cultural contact scripts — Greek (heavy cultural exchange), Phoenician (Pyrgi bilingual), Rhaetic/Lemnian (Tyrrhenian family?)
- Tier 3 (20% weight): Operator scripts (if available; redistribute if not)
- Tier 4 (20% weight): Proven methodology scripts — combinatorial/contextual methods validated on similar partially-deciphered isolates
- **Key**: The Pyrgi bilingual + Latin/Greek loanword evidence are primary anchors

### PHASE 3: Vocabulary Expansion (Adapted)
- Start from known: numbers (1-6 confirmed), kinship terms, magistrate titles, divine names
- Funerary formula analysis: name + patronymic + age + titles (formulaic = expandable)
- The Liber Linteus as ritual calendar: dates + rituals + offerings (contextually constrained)
- Etruscan words borrowed into Latin (many are known) provide back-testing
- Glosses in ancient authors (Varro, Festus, etc.) as secondary evidence
- Output: Expanding vocabulary with confidence levels

### PHASE 4: Grammatical Expansion (Adapted)
- Agglutinative tendencies confirmed (case suffixes, possessives)
- Cases: nominative (-Ø), genitive (-s/-l), dative/locative (-i/-si)
- Verbal system: partially mapped (past -ce, active/passive distinctions)
- Complex syntax remains challenging (subordination, conditionals)
- Output: Extended grammar beyond current academic consensus

### PHASE 5-8: Cultural/Historical Context
- Phase 5: Etruscan religion (haruspicy, sacred calendar, divine hierarchy)
- Phase 6: Political system (zilath, lucumo, lauchme titles)
- Phase 7: Trade vocabulary (via archaeological evidence of goods)
- Phase 8: Etruscan-Roman interaction (institutional borrowings both directions)

### PHASE 9-12: Advanced Decipherment
- Use combinatorial analysis on the Liber Linteus (longest continuous text)
- Extract vocabulary from Tabula Capuana (tile calendar, related to Liber Linteus)
- Test proposed meanings across ALL attestations
- Build from formulaic/constrained texts outward into free text
- Exploit Latin borrowings from Etruscan as confirmed semantic data

### PHASE 13-16: Validation
- Every proposed word meaning must work in all attested contexts
- Cross-reference with known Latin borrowings from Etruscan
- Pyrgi bilingual alignment verification
- Archaeological consistency (do proposed readings match material culture?)

### PHASE 17-20: Status
- Currently at: ~20-25% vocabulary (academic consensus); higher with speculative work
- Best path: systematic formulaic text analysis + Liber Linteus exhaustive study
- Barrier: Language isolate status limits comparative approach
- Opportunity: Large corpus (13,000 inscriptions) provides statistical power
