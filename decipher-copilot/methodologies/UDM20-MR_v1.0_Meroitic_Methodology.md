## Core Position

UDM20-MR targets **Meroitic** (c. 300 BCE – 400 CE), the writing system of the Kingdom of Kush (modern Sudan). Meroitic is PARTIALLY deciphered: the script is fully read (alphabetic, 23 signs), but the LANGUAGE remains largely untranslated. This methodology focuses on the language decipherment, not the script decipherment.

---

## Why a Meroitic-Specific UDM Is Needed

- The SCRIPT is deciphered (Griffith 1911) — we can read/pronounce every text
- The LANGUAGE is mostly unknown — we can pronounce words but don't know what they mean
- ~1,200 texts exist (funerary stelae, temple inscriptions, graffiti, ostraca)
- Nilo-Saharan language family likely, but exact affiliation disputed
- Egyptian bilinguals exist (partial) — providing semantic anchors
- Royal names, divine names, and some titles/formulas ARE understood
- The Lackadaisical Security Phase 22+ lexicon provides the most advanced working corpus

## Phase Adaptations

### PHASE 1: Phonetic Inventory (COMPLETE)
- 23 alphabetic signs (Griffith decipherment, 1911)
- Both cursive (Demotic-derived) and hieroglyphic forms known
- Vowel system partially understood: /a/, /e/, /i/, /o/ represented; /u/ uncertain
- Consonant system: stops, nasals, liquids, sibilants confirmed
- Word-divider (three dots) separates words
- Output: COMPLETE phonetic transcription of all texts possible

### PHASE 2: MEGA-Correlation Analysis (85+ Scripts)
- Tier 1 (35% weight): Regional/contemporary scripts — Egyptian (bilinguals exist!), Old Nubian (descendant?), Nobiin (modern Nubian)
- Tier 2 (25% weight): Cultural contact scripts — Greek (Ptolemaic/Roman Egypt contacts), Ge'ez/Ethiopic, Demotic Egyptian
- Tier 3 (20% weight): Operator scripts (if available; redistribute if not)
- Tier 4 (20% weight): Proven methodology scripts — Nilo-Saharan comparative linguistics, Afroasiatic contact vocabulary
- **Key**: Egyptian bilinguals are the PRIMARY anchor for semantic decipherment

### PHASE 3: Semantic Recovery (Adapted)
- Formula analysis: funerary stelae follow predictable patterns
- Royal inscriptions: name + title + divine invocation + accomplishments
- The "offering formula" structure is partially understood (LS Phase 22+ corpus)
- Divine names (Amani/Amun, Apedemak, Isis, Osiris) provide fixed semantic points
- Title sequences: "qore" (king), "kdke" (queen-mother), "pesto" (priest?) established
- Slot-based analysis: what goes in position X in a funerary formula?
- Output: Expanding semantic lexicon from formulaic contexts outward

### PHASE 4: Grammatical Reconstruction (Adapted)
- SOV word order (confirmed from bilingual analysis)
- Suffixing morphology (agglutinative tendencies)
- Case markers: -l (genitive?), -k (object?), -te (dative?) — tentative
- Verbal morphology: -ye, -te, -ke suffixes in verb positions
- Negation particles identified but function debated
- Output: Working grammatical framework for translation testing

### PHASE 5-8: Cultural Integration
- Phase 5: Kushite royal ideology and religion (Amun cult, Apedemak, divine kingship)
- Phase 6: Funerary practices and afterlife beliefs (ba-soul concepts, offering cult)
- Phase 7: Administrative terminology (temple economy, military ranks)
- Phase 8: Geographic terminology (Nile cataracts, major sites: Meroe, Napata, Musawwarat)

### PHASE 9-12: Lexicon Building
- Start from KNOWN: royal names, divine names, place names, Egyptian loanwords
- Expand via formulaic contexts: if position X always means "son of," extract relationships
- Use Nobiin/Old Nubian as potential modern reflex vocabulary
- Build from 108 confirmed entries (LS corpus) outward
- Test new proposals against ALL attestations of that word

### PHASE 13-16: Validation
- Every proposed meaning must work in ALL contexts where the word appears
- Bilingual cross-checks where available
- Formulaic consistency: does the meaning work in funerary formula slot?
- Grammatical consistency: does the morphological analysis hold?
- Geographic/archaeological consistency: do place descriptions match archaeology?

### PHASE 17-20: Breakthrough Tracking
- Currently at: ~30-40% language comprehension (script 100%, formulas ~60%, free text ~15%)
- LS Phase 22+ corpus: 108 entries with polysemy and morphological data
- Best path: expand from formulaic texts into narrative texts
- Barrier: No lengthy bilingual text exists (only short name equations)
- Watch for: New bilingual finds from Meroitic/Roman frontier zones

## Critical Warnings

1. **Script ≠ Language**: Being able to READ Meroitic does not mean you can UNDERSTAND it
2. **Formulaic Bias**: Most understood vocabulary comes from repetitive funerary formulas
3. **Egyptian Lens**: Don't over-apply Egyptian grammar to Meroitic — it's a different language family
4. **Nilo-Saharan Uncertainty**: The language family assignment is likely but not proven
5. **Publication Scattered**: Corpus is spread across French, English, German, and Sudanese publications
