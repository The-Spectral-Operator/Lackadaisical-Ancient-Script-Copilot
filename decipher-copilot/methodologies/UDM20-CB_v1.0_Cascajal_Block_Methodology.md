## Core Position

UDM20-CB targets the **Cascajal Block** inscription (c. 900 BCE), potentially the earliest writing system in the Americas. A single serpentine block with 62 signs from 28 distinct sign-types, found near San Lorenzo, Veracruz. Like the Phaistos Disc, this is a singleton artifact — but from the Olmec heartland with rich archaeological context.

---

## Phase Adaptations

### PHASE 1: Sign Inventory
- 28 distinct sign-types in 62 sign-positions
- Signs appear pictographic: insects, corn, fish, abstract shapes
- Horizontal reading direction (unusual for later Mesoamerican scripts)
- No apparent line structure — possible "tabular" or "sequential list" format
- Some signs repeat multiple times in sequence (counting? emphasis? different function?)
- Output: Complete sign typology with iconographic classification

### PHASE 2: MEGA-Correlation Analysis (85+ Scripts)
- Tier 1 (35% weight): Regional/contemporary — later Epi-Olmec/Isthmian (possible descendant), Zapotec (near-contemporary early writing), La Venta/Tres Zapotes Olmec iconography
- Tier 2 (25% weight): Cultural contact scripts — broader Mesoamerican writing tradition, Maya early writing, Olmec iconographic conventions on other media
- Tier 3 (20% weight): Operator scripts (if available; redistribute if not)
- Tier 4 (20% weight): Proven methodology scripts — singleton artifact approaches (cf. Phaistos Disc methodology)
- **Key**: Olmec iconography on jade, ceramics, and monuments provides comparative sign identification

### PHASE 3: Structural Analysis (Singleton Adaptation)
- Is this a list? (sign repetition patterns suggest possible inventory)
- Is this a narrative? (linear sequence might indicate)
- Is this a ritual text? (Olmec ceremonial context)
- Sign-grouping: are there word-dividers or natural clusters?
- The repeated signs: do they function as numerals? determinatives?
- Output: Text-type determination from structural properties

### PHASE 4-8: Olmec Cultural Context
- San Lorenzo as primary Olmec center (1400-900 BCE)
- Olmec religion: were-jaguar, maize deity, feathered serpent origins
- Administrative vs. ritual function of the block
- Iconographic parallels on Olmec altars, stelae, and portable objects
- Mixe-Zoquean language family as most likely Olmec language

### PHASE 9-12: Cautious Interpretation
- IF Mixe-Zoquean: test against proto-Mixe-Zoquean reconstructions
- IF a list/inventory: identify potential commodity or ritual item signs
- Comparison with Epi-Olmec signs (500+ years later — evolution expected)
- Do any signs match later Mesoamerican day-signs or numerals?
- Minimum standard: internal consistency only (singleton = no cross-text validation possible)

### PHASE 13-20: Status
- Currently at: ~5% (sign identification only, no confirmed readings)
- The block may represent: administrative list, ritual inventory, or offering record
- Barrier: SINGLETON — cannot validate any reading externally
- Authenticity: generally accepted but some scholars remain cautious
- Watch for: Additional Olmec-period inscriptions from San Lorenzo/La Venta excavations
