### By: Lackadaisical Security - Linguistics Division

## Core Position

UDM20-SK targets **Thracian inscriptions**, including the Sitovo inscription (Bulgaria), the Ezerovo ring, the Duvanli inscriptions, and other scattered texts potentially in the Thracian language (c. 600 BCE – 100 CE). Most use the Greek alphabet, but reading them as Thracian (an extinct Indo-European language) remains challenging due to minimal vocabulary knowledge.

---

## Phase Adaptations

### PHASE 1: Corpus Assembly
- ~20-30 inscriptions potentially in Thracian
- Most written in Greek alphabet (script = readable, language = problem)
- Ezerovo ring: longest continuous Thracian text (~8 words)
- Sitovo inscription: rock-cut text, Greek letters, debated reading
- Duvanli artifacts: short inscriptions on gold/silver
- Thracian words preserved in Greek/Latin authors (~30 glosses)
- Output: Unified corpus with paleographic standardization

### PHASE 2: MEGA-Correlation Analysis (85+ Scripts)
- Tier 1 (35% weight): Regional/contemporary — Greek (contact language, same alphabet), Phrygian (nearest IE relative?), Dacian (same Paleo-Balkan branch?)
- Tier 2 (25% weight): Cultural contact scripts — Latin, Celtic (La Tène contacts), Illyrian (neighboring Paleo-Balkan)
- Tier 3 (20% weight): Operator scripts (if available; redistribute if not)
- Tier 4 (20% weight): Proven methodology scripts — Indo-European comparative method (the standard tool for fragmentarily attested IE languages)
- **Key**: Indo-European comparative linguistics is the PRIMARY tool — Thracian IS Indo-European

### PHASE 3: Linguistic Reconstruction
- Thracian is Indo-European (confirmed by place names, personal names, glosses)
- Sub-grouping debated: Daco-Thracian? Thraco-Phrygian? Paleo-Balkan isolate?
- ~30 glossed words (ancient author citations) provide minimal vocabulary
- Hundreds of personal names and place names provide phonological data
- Albanian possibly descends from Thracian (or Illyrian) — potential modern reflex
- Output: Phonological and morphological reconstruction from onomastic data

### PHASE 4-8: Contextual Framework
- Phase 5: Thracian kingdom(s) and tribal structure
- Phase 6: Greek colonial contacts (Thracian coast = heavily colonized)
- Phase 7: Religious practices (Dionysus/Sabazios, Orphic traditions)
- Phase 8: Material culture context for inscribed objects

### PHASE 9-12: Reading Strategy
- Apply Greek alphabet values (straightforward)
- Parse resulting phoneme strings against IE morphological expectations
- Test against known Thracian onomastic patterns (name element identification)
- Use glossed vocabulary as confirmed anchor points
- Look for: verbal forms, case endings, particles matching IE expectations

### PHASE 13-20: Status
- Currently at: ~10-15% (phonology mapped from names, very limited textual understanding)
- Barrier: Tiny textual corpus, language barely attested
- Best path: More inscriptions needed + Albanian comparative data
- Watch for: New inscriptions from Bulgarian/Greek Thracian tomb excavations
