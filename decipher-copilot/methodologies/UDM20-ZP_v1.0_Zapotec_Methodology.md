## Core Position

UDM20-ZP targets the **Zapotec script** (c. 600 BCE – 800 CE), one of Mesoamerica's earliest writing systems from the Valley of Oaxaca. The script appears on monuments, ceramics, and bones, and likely encodes an ancestral Zapotec language. Calendar names and some logograms are partially understood, but the full system remains undeciphered.

## Phase Adaptations

### PHASE 1: Sign Inventory
- ~200 signs documented (Urcid 2001 corpus)
- System type: likely logosyllabic (Mesoamerican tradition)
- Key sites: Monte Albán, San José Mogote, Dainzú
- Period spans 1,400 years — expect significant evolution
- Calendar signs (260-day cycle) provide fixed reference points
- Output: Chronologically stratified sign inventory

### PHASE 2: MEGA-Correlation Analysis (85+ Scripts)
- Tier 1 (35% weight): Regional/contemporary — Maya (deciphered, same Mesoamerican tradition), Epi-Olmec/Isthmian, Mixtec
- Tier 2 (25% weight): Cultural contact scripts — Teotihuacan notation, early Aztec, broader Mesoamerican writing tradition
- Tier 3 (20% weight): Operator scripts (if available; redistribute if not)
- Tier 4 (20% weight): Proven methodology scripts — Maya decipherment success provides methodological template
- **Key**: Maya script structural analogy + modern Zapotec languages for linguistic back-projection

### PHASE 3: Linguistic Framework
- Language: ancestral Zapotecan (Otomanguean family)
- Modern Zapotec languages (~60+) provide comparative vocabulary
- Tonal language: script may NOT encode tone (common Mesoamerican limitation)
- Calendar system partially shared with broader Mesoamerica
- Expected content: royal history, conquests, ritual calendrics, genealogy
- Output: Linguistic model constraints

### PHASE 4-8: Archaeological Context
- Monte Albán as primary political/ritual center
- "Danzante" slabs: captive/conquest records (earliest texts)
- Building J: conquest stelae with place names and dates
- Genealogical registers on South Platform
- Rich archaeological context provides semantic constraints

### PHASE 9-12: Decipherment Strategy
- Start from calendar signs (partially known via Mesoamerican comparisons)
- Use place-name glyphs (topographically verifiable)
- Royal name sequences (genealogical context constrains meaning)
- Apply Maya-style logosyllabic analysis as structural hypothesis
- Test proposed readings against Zapotecan linguistic reconstruction

### PHASE 13-20: Status
- Currently at: ~15-20% (calendar, some logograms, basic text structure)
- Best path: Maya-style syllabic substitution testing + modern Zapotec vocabulary
- Barrier: Long time span with script evolution, no bilingual
- Watch for: New Monte Albán finds, improved proto-Zapotec reconstruction
