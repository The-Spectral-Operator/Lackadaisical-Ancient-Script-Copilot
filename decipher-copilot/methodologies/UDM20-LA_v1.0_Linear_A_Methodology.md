## Core Position

UDM20-LA is designed for **Linear A**, the undeciphered writing system of Minoan Crete (c. 1800–1450 BCE). Linear A shares its sign repertoire partially with the deciphered Linear B (Mycenaean Greek), making comparative structural analysis uniquely productive while also uniquely dangerous (false cognate traps).

---

## Why a Linear A-Specific UDM Is Needed

- Linear B provides a direct structural comparand, but the underlying language is NOT Greek
- The corpus is moderate (~1,500 inscriptions, ~7,400 sign-groups)
- Administrative tablets dominate, but religious/votive texts offer different vocabulary
- Place names (toponyms) shared with Linear B provide anchoring points
- The Minoan language is an isolate — no confirmed language family

## Phase Adaptations

### PHASE 1: Sign Inventory (Adapted)
- Use Godart-Olivier GORILA corpus as primary sign reference
- Cross-map with Linear B sign forms (AB series equivalences)
- Identify signs unique to Linear A (no Linear B equivalent)
- **Key**: Do NOT assume shared form = shared phonetic value
- Output: ~90 syllabic signs, ~100 logograms cataloged

### PHASE 2: MEGA-Correlation (Adapted)
- Tier 1 (40% weight): Linear B structural parallels
- Tier 2 (25% weight): Cypro-Minoan, Cretan Hieroglyphic
- Tier 3 (20% weight): Anatolian scripts (Luwian Hieroglyphs, Lycian)
- Tier 4 (15% weight): Semitic scripts (Ugaritic, early Phoenician)
- **Critical filter**: Shared signs ≠ shared language. Weight STRUCTURAL parallels over phonetic assumptions.

### PHASE 3: Semantic Clustering (Adapted)
- Administrative formula detection ("total" signs, numeric tallies)
- Votive formula detection (recurring sequences on libation tables)
- Geographic marker detection (compare with Linear B toponyms)
- The "libation formula" (A-TA-I-*301-WA-JA / JA-SA-SA-RA-ME variants) is the primary anchor
- Output: Semantic fields without forced translation

### PHASE 4: Proto-Grammatical Structure (Adapted)
- Identify prefixing vs. suffixing behavior
- Map case-like markers from administrative contexts
- Verbal morphology from votive repetition patterns
- **Do NOT impose Greek grammar** — Linear A language is NOT Indo-European (probably)
- Agglutinative hypothesis vs. isolating hypothesis both tested

### PHASE 5-8: Cultural Integration
- Phase 5: Minoan palace economy reconstruction (Knossos, Phaistos, Hagia Triada, Zakros)
- Phase 6: Ritual practice inference from libation table distributions
- Phase 7: Trade network analysis (Egyptian, Anatolian, Levantine contacts)
- Phase 8: Peak sanctuary inscriptions as independent verification corpus

### PHASE 9-12: Phonetic Testing
- Use Linear B cognate signs as HYPOTHESIS ONLY
- Test against toponyms preserved in both scripts
- The "Minoan substrate" words in Linear B Greek provide back-testing anchors
- Place names: KA-TA-NA, KU-DO-NI-JA, A-MI-NI-SO, etc.
- Validate against Eteocretan inscriptions (alphabetic Minoan continuity)

### PHASE 13-16: Validation
- Cross-check with GORILA frequency data
- Test hypotheses against newly discovered inscriptions
- Statistical rigor: minimum 3 independent attestations for any phonetic value
- Peer-reviewed publication cycle required before "confirmed" status

### PHASE 17-20: Breakthrough Tracking
- Currently at ~15-20% decipherment (numerals + some logograms confirmed)
- Phonetic grid: 30-40% hypothetical (via Linear B transfer)
- Language identification: 0% confirmed (leading hypotheses: Tyrrhenian, Anatolian substrate, isolate)
- Track all hypotheses with explicit confidence bands

## Critical Warnings

1. **The Greek Trap**: Just because a sign looks like Linear B "da" does NOT mean it says /da/ in Linear A
2. **Small Corpus Bias**: 7,400 sign-groups is not statistically robust for complex grammatical claims
3. **Administrative Skew**: Most texts are accounts — vocabulary is heavily biased toward commodities
4. **Publication Lag**: New inscriptions from ongoing excavations may change statistics significantly
