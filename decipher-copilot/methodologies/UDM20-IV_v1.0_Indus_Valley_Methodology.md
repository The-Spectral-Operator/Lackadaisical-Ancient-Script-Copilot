## Core Position

UDM20-IV targets the **Indus Valley Script** (Harappan script, c. 2600–1900 BCE), one of the most contested undeciphered writing systems. With ~4,200 inscriptions averaging only 4-5 signs each, and no bilingual, the script presents extreme challenges that require a methodology emphasizing statistical pattern analysis over speculative phonetic assignment.

---

## Why an Indus-Specific UDM Is Needed

- Extremely short average text length (4-5 signs) makes linguistic analysis fragile
- No bilingual or translation key exists
- The underlying language is unknown (Dravidian, Indo-Aryan, language isolate, or non-linguistic all proposed)
- Whether it represents full writing, proto-writing, or a non-linguistic symbol system is still debated
- Geographic spread (1,500+ km) means potential dialect/script variation
- Seal context vs. pottery marks vs. tablets may represent different register/systems

## Phase Adaptations

### PHASE 1: Sign Inventory (Adapted)
- Use Mahadevan (1977) concordance as primary reference + Wells (2015) digital extensions
- ~417 distinct signs (conservative) vs ~600+ (liberal counting)
- Allograph identification critical: distinguish signs vs. variants
- Positional statistics: initial, medial, final distributions per sign
- Directionality confirmation (right-to-left primary)
- Output: Cleaned inventory with frequency ranks and positional profiles

### PHASE 2: MEGA-Correlation Analysis (85+ Scripts)
- Tier 1 (35% weight): Regional/contemporary scripts — Proto-Elamite, early Sumerian proto-cuneiform, Dilmun seals, BMAC symbols
- Tier 2 (25% weight): Cultural contact scripts — Mesopotamian cuneiform (Akkadian trade contacts), early Dravidian scripts (Brahmi descendants as back-projection), Old Persian
- Tier 3 (20% weight): Operator scripts (if available; otherwise redistribute to Tiers 1 & 2)
- Tier 4 (20% weight): Proven methodology scripts — apply statistical/structural methods validated on other decipherments
- **Key**: Indus IS illuminated by regional parallels, especially Mesopotamian trade seals and Proto-Elamite administrative structures

### PHASE 3: Semantic Clustering (Adapted)
- Seal inscription patterns (owner marks, titles, institutional labels)
- Positional analysis: "fish" sign variants at position 2-3, "jar" signs at final position
- Numerical system (short strokes) partially understood
- The "U+fish+number" pattern as potential title formula
- Direction-of-reading tests via sign-pair frequency analysis
- Output: Functional categories WITHOUT phonetic assignment

### PHASE 4: Proto-Grammatical Structure (Adapted)
- Test for agglutinative morphology (Dravidian-compatible)
- Test for inflectional morphology (Indo-Aryan-compatible)
- Test for isolating/classificatory system (non-IE, non-Dravidian)
- Positional grammar: what position constraints exist?
- **The "is it writing?" question**: Apply conditional entropy tests (Rao et al. 2009) to establish whether sign sequences have linguistic structure
- Output: Structural profile without language-family commitment

### PHASE 5-8: Cultural Integration
- Phase 5: Harappan urban planning and administrative systems
- Phase 6: Trade seal functions (Mesopotamian/Gulf contacts)
- Phase 7: Workshop marks vs. institutional marks vs. personal names
- Phase 8: Geographic distribution patterns (Mohenjo-daro vs. Harappa vs. peripheral sites)

### PHASE 9-12: Hypothesis Testing
- DO NOT assign phonetic values without multiple convergent evidence lines
- Test Parpola's Dravidian readings against statistical constraints
- Test Farmer-Sproat-Witzel's non-linguistic hypothesis against entropy data
- Test structural patterns against known short-inscription scripts (potter's marks, ownership marks)
- Minimum threshold: 5+ independent attestations in consistent context before tentative phonetic value

### PHASE 13-16: Validation
- Every proposed reading must survive:
  - Positional frequency validation
  - Contextual consistency check (does it make sense on seals?)
  - Archaeological context compatibility
  - Statistical significance above random
- No single reading accepted without cross-site attestation

### PHASE 17-20: Breakthrough Tracking
- Currently at: ~5% decipherment (numerical system partially confirmed)
- Major barrier: No bilingual, no confirmed language family
- Best paths forward: computational statistical analysis + new corpus discoveries
- Watch for: Rakhigarhi DNA implications, new Dholavira reservoir inscription analysis

## Critical Warnings

1. **The Length Problem**: Average 4-5 signs is TOO SHORT for reliable linguistic analysis by traditional methods
2. **The Language Problem**: Without knowing the language, phonetic values are speculation
3. **The Writing Problem**: The system may be PARTIALLY non-linguistic (emblematic/heraldic)
4. **Cherry-Picking Risk**: With 417+ signs and no constraints, almost any language can be "read" into the data
5. **Publication Bias**: Dramatic "decipherment" claims get published; null results don't
