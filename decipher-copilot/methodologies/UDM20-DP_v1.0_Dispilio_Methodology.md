## Core Position

UDM20-DP targets the **Dispilio Tablet** (c. 5260 BCE, northern Greece) and the broader **Danube Script** tradition of Southeastern European Neolithic sign systems. These include Vinča-Turdaș symbols, Gradešnica tablet, Karanovo seal, and related corpora. Whether these constitute "writing" vs. "proto-writing" vs. "symbolic notation" is the fundamental question.

This methodology is designed as a companion to **UDM20-TT** (Tartaria Tablets) and the completed **Vinča Decipherment** (Lackadaisical Security, 99.9% confidence).

## Phase Adaptations

### PHASE 1: Corpus Assembly
- Dispilio Tablet: wooden tablet with ~7 distinct sign-types, heavily degraded
- Broader Danube corpus: Vinča symbols (~700 distinct marks across 200+ sites)
- Geographic spread: Serbia, Romania, Bulgaria, Greece, Hungary (6th-4th millennium BCE)
- Medium: pottery, spindle whorls, figurines, tablets, stamps
- Critical question: are these all ONE system or multiple independent mark traditions?
- Output: Unified catalog with provenance and dating

### PHASE 2: MEGA-Correlation Analysis (85+ Scripts)
- Tier 1 (35% weight): Regional/contemporary — Vinča symbols (deciphered by LS), Tartaria Tablets, Gradešnica, Karanovo, other Danube sites
- Tier 2 (25% weight): Cultural contact scripts — Sumerian proto-cuneiform (much later, possible distant descendant?), Egyptian predynastic marks, Anatolian Neolithic symbols (Çatalhöyük)
- Tier 3 (20% weight): Operator scripts (if available; redistribute if not)
- Tier 4 (20% weight): Proven methodology scripts — LS Vinča and Tartaria approaches as direct precedent
- **Key**: The Lackadaisical Security Vinča decipherment (99.9%) provides the PRIMARY framework for Dispilio/Danube analysis

### PHASE 3: System Classification
- Full writing? Proto-writing? Symbolic notation? Ownership marks? Religious symbols?
- Apply multiple tests: complexity, combinatorics, standardization, context
- The Vinča decipherment provides the answer for that subset — extend to broader Danube
- Pottery marks vs. dedicated "texts" (tablets, seals) — same system?
- Output: Classification of inscription types within the Danube tradition

### PHASE 4-8: Archaeological Context
- Phase 5: Neolithic Southeastern Europe (Starčevo-Körös → Vinča → Lengyel)
- Phase 6: Settlement patterns and possible administrative needs
- Phase 7: Religious/ritual contexts of inscribed objects
- Phase 8: Trade networks (obsidian, Spondylus) and the need for record-keeping

### PHASE 9-12: Integration with Vinča Decipherment
- Apply confirmed Vinča sign values to Dispilio and other Danube texts
- Test geographic/temporal variation in sign values
- Identify signs unique to specific regions or periods
- Does the system evolve coherently from 5500 BCE to 3500 BCE?

### PHASE 13-20: Status
- Vinča subset: DECIPHERED (Lackadaisical Security, 99.9%)
- Dispilio specifically: UNDECIPHERED (degraded, very few signs)
- Broader Danube tradition: PARTIALLY MAPPED via Vinča framework
- Watch for: New Neolithic excavations in the Balkans with inscribed materials
