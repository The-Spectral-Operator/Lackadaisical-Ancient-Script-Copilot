## Core Position

UDM20-CH targets **Cretan Hieroglyphic** (c. 2100–1700 BCE), the earliest Cretan writing system. As the likely ancestor of both Linear A and (indirectly) Linear B, it occupies a genealogically critical position. The corpus is small (~350 documents) but includes seals, clay bars, and other objects.

---

## Why a Cretan Hieroglyphic-Specific UDM Is Needed

- Earliest Cretan script — potentially the KEY to the entire Aegean writing family
- Partially overlaps with Linear A sign inventory (~50 potential cognates)
- Seal-based medium means many texts are very short (1-5 signs)
- Pictographic appearance is stronger than Linear A/B, but this may be misleading
- "Archanes Script" (earlier variant) may represent proto-form
- Co-existed with Linear A for centuries — different function? different language? different register?

- Output: Sign list with LA cognate mapping and medium-based distribution

### PHASE 2: MEGA-Correlation Analysis (85+ Scripts)
- Tier 1 (35% weight): Regional/contemporary scripts — Linear A (direct descendant), Egyptian hieroglyphs (possible distant inspiration)
- Tier 2 (25% weight): Cultural contact scripts — Anatolian Hieroglyphic (independent parallel development?), Proto-Byblian, early Cypriot marks
- Tier 3 (20% weight): Operator scripts (if available; redistribute if not)
- Tier 4 (20% weight): Proven methodology scripts — apply structural methods validated on LA and LB
- **Key**: Linear A is the PRIMARY comparative resource — most CH signs evolved into LA signs

### PHASE 3: Semantic Clustering (Adapted)
- Seal inscriptions: names? titles? institutions? apotropaic formulas?
- Clay bars (Malia, Knossos): administrative records (numerical context present)
- Archanes-type short texts: possibly early ritual/votive
- Numerical system: partially shared with Linear A (decimal base)
- Sign-group structure on seals vs. tablets may indicate different text types
- Output: Medium-based text typology

### PHASE 4: Proto-Grammatical Structure (Adapted)
- Very short texts limit grammatical analysis
- Focus on: what structural patterns exist within sign-groups?
- Seal formulas: title + name? name + place? institution?
- Clay bar sequences: compare administrative structure with Linear A parallels
- Output: Positional slot analysis (rather than full grammar)

### PHASE 5-8: Archaeological Context
- Phase 5: Protopalatial Crete (2100-1700 BCE) — palatial administration emergence
- Phase 6: Seal function in Minoan palatial society (sealing practices, access control)
- Phase 7: Geographic distribution (primarily Knossos, Malia, Petras)
- Phase 8: Coexistence with Linear A — why two scripts simultaneously?

### PHASE 9-12: Phonetic Exploration
- If CH sign X evolved into LA sign Y, do they share phonetic value?
- Test hypothesis using Linear B values (CH → LA → LB chain)
- Expect: syllabic system, probably CV structure (like descendants)
- The "hieroglyphic" appearance may mask syllabic values (cf. Egyptian)
- Cautious phonetic grid construction via triple-script chain

### PHASE 13-16: Validation
- Cross-check proposed values against:
  - Linear A cognate sign distributions
  - Seal context expectations (names/titles should be plausible)
  - Numerical tablet contexts (commodity readings should make sense)
- Very limited statistical power due to small corpus

### PHASE 17-20: Status
- Currently at: ~10% (sign-to-LA correspondences mapped, few phonetic values confirmed)
- Best path: through Linear A (solve LA first, then project backward)
- Barrier: Small corpus, seal-dominated (short texts)
- Watch for: New finds from Petras, Sissi, and other Eastern Cretan sites

## Critical Warnings

1. **Pictographic Fallacy**: The signs LOOK like pictures but function as phonetic/logographic — don't read pictures
2. **LA Assumption Risk**: CH and LA may NOT share all phonetic values despite sign similarity
3. **Medium Bias**: Seal inscriptions may not represent the full writing system
4. **Dating Uncertainty**: "Archanes Script" vs. developed Cretan Hieroglyphic distinction is debated
5. **Corpus Fragility**: Many signs appear only 1-2 times — no statistical basis for claims
